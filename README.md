# ニュータンタンメン店舗口コミ分析

## 概要
ニュータンタンメン店舗の口コミデータを収集・分析し、
KMeans クラスタリングで店舗タイプを分類するプロジェクトです。

## 分析目的
- 口コミテキストから店舗の特徴を抽出
- KMeans で店舗タイプを分類（例：辛さ重視、ボリューム重視）
- 分類結果から店舗の傾向を可視化・解釈

## 使用技術
- Python
- pandas, numpy
- scikit-learn
- 日本語NLP（janome/fugashi）
- SQL (SQLite)
- Matplotlib / Seaborn / Plotly

## データ取得
- 公式API「ぐるなびAPI（日本語版 Ver3）」を使用
- 利用規約に基づき取得しています
- 口コミテキストや評価、緯度経度情報を使用

## 分析手順
1. APIから店舗データを取得
2. 口コミデータを前処理（形態素解析・不要語除去）
3. TF-IDFでテキストをベクトル化
4. KMeansクラスタリングで店舗タイプを分類
5. クラスタごとの特徴語・平均評価・地図可視化
6. 結果をまとめてレポート化
